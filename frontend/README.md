# React + Vite CRUD UI

Frontend for the FastAPI CRUD demo.

## Setup

```bash
cd frontend
npm install
```

## Environment

Create a `.env` file in `frontend/`:

```
VITE_API_URL=http://localhost:8000
```

If unset, it defaults to `http://localhost:8000`.

## Run

```bash
npm run dev -- --host --port 5173
```

Visit http://localhost:5173.
